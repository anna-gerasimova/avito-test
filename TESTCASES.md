# Задание 2

## Тест-кейс 1. Базовое создание задачи

1. **Название:** Создание задачи с минимальными данными.
2. **Предусловия:** Пользователь авторизован на платформе.
3. **Шаги:**
   1. Перейти на страницу таск-трекера ([https://avito-tech-internship-psi.vercel.app/issues](https://avito-tech-internship-psi.vercel.app/issues)).
   2. Нажать кнопку «Создание задачи».
   3. Заполнить только обязательные поля (например, название, проект, приоритет, исполнитель).
   4. Нажать кнопку «Создать».
4. **Ожидаемый результат:** Новая задача успешно создана, отображается в списке задач с указанным заголовком.
5. **Статус:** пройден/не пройден

## Тест-кейс 2. Проверка валидации обязательных полей

1. **Название:** Попытка создания задачи без заполнения обязательных полей.
2. **Предусловия:** Пользователь авторизован на платформе.
3. **Шаги:**
   1. Перейти на страницу таск-трекера ([https://avito-tech-internship-psi.vercel.app/issues](https://avito-tech-internship-psi.vercel.app/issues)).
   2. Нажать кнопку «Создание задачи».
   3. Опустить заполнение одного или нескольких обязательных полей.
   4. Нажать кнопку «Создать».
4. **Ожидаемый результат:** Система выдаёт ошибку, или предупреждение о необходимости заполнения обязательных полей, или кнопка «создать» не кликабельна. Задача не создаётся.
5. **Статус:** пройден/не пройден

## Тест-кейс 3. Проверка уникальности заголовков задач

1. **Название:** Попытка создания задачи с дублирующимся заголовком.
2. **Предусловия:** На платформе уже существует задача с определённым заголовком. Пользователь авторизован.
3. **Шаги:**
   1. Перейти на страницу таск-трекера ([https://avito-tech-internship-psi.vercel.app/issues](https://avito-tech-internship-psi.vercel.app/issues)).
   2. Нажать кнопку «Создание задачи».
   3. Ввести заголовок, идентичный уже существующей задаче (например, Реализация новой галереи изображений).
   4. Нажать кнопку «Создать».
4. **Ожидаемый результат:** Система либо позволяет создать задачу с дублирующимся заголовком, либо выдаёт предупреждение о дублировании (в зависимости от логики платформы).
5. **Статус:** пройден/не пройден

## Тест-кейс 4. Базовое открытие карточки задачи

1. **Название:** Открытие карточки существующей задачи.
2. **Предусловия:** Пользователь авторизован, в системе есть созданные задачи.
3. **Шаги:**
   1. Перейти на страницу таск-трекера ([https://avito-tech-internship-psi.vercel.app/issues](https://avito-tech-internship-psi.vercel.app/issues)).
   2. Нажать (выбрать) на любую задачу в списке задач (например, Реализация новой галереи изображений).
   3. Дождаться загрузки карточки задачи.
4. **Ожидаемый результат:** Открылась карточка задачи с полной информацией — название, описание, проект, приоритет и другие поля. Интерфейс карточки загружен без ошибок.
5. **Статус:** пройден/не пройден

## Тест-кейс 5. Обновление готовой карточки задачи

1. **Название:** Обновление информации в готовой карточке задачи.

3. **Шаги:**
   1. Перейти на страницу таск-трекера ([https://avito-tech-internship-psi.vercel.app/issues](https://avito-tech-internship-psi.vercel.app/issues)).
   2. Нажать (выбрать) на любую задачу в списке задач (например, Реализация новой галереи изображений).
   3. Скорректировать данные в задаче (например, поменять название «Реализация старой галереи изображений»).
   4. Нажать кнопку «Обновить».
4. **Ожидаемый результат:** Задача успешно обновлена, отображается в списке задач с новым указанным заголовком.
5. **Статус:** пройден/не пройден
